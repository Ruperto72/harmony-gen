# 🎵 HarmonyGen

**HarmonyGen** är en interaktiv webbapplikation som hjälper sångare att hitta och skapa stämmor i realtid. Genom att analysera din röst via mikrofonen föreslår appen musikaliskt korrekta harmonier baserat på valda intervall.

## 🚀 Funktioner
* **Realtidsanalys:** Identifierar tonhöjd (pitch) och notnamn direkt när du sjunger.
* **Smart Harmonisering:** (Under utveckling) Skapar stämmor som följer melodin i realtid.
* **Inställningspanel:** Anpassa harmonityp (terser, kvinter) via en dedikerad kontrollpanel.
* **Web-baserad:** Inga installationer krävs – körs direkt i webbläsaren via Web Audio API.

## 🛠 Teknikstack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+).
* **Ljudmotor:** [Tone.js](https://tonejs.github.io/) för avancerad ljudhantering.
* **Tonhöjdsdetektering:** [Pitchy](https://github.com/TobiStrummer/pitchy) för exakt frekvensanalys.
* **Hosting:** Utbyggd via GitHub Pages.
